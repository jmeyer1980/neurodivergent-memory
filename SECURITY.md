# Security Policy

## Supported Versions

Security fixes are provided for the latest published release.

## Reporting a Vulnerability

If you discover a security issue:

1. Prefer opening a private GitHub Security Advisory in this repository.
2. If advisory tooling is unavailable, open a GitHub issue and clearly label it as a security concern.

Please do not post exploit details publicly before maintainers have a chance to assess and respond.

## Contact

Security communication for this project is handled through GitHub only.
No email contact is provided.
