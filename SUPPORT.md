# Support

## Getting Help

Use GitHub for all support:

- Open an issue for bugs, docs gaps, and feature requests.
- Use pull request comments for code-level discussions.

Project links:

## Pricing and Access

- This project is free to use.
- No API key is required.
- Any costs from external services (for example your own Docker Hub or npm account usage) are outside this project.
